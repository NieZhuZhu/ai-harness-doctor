# Support

## Questions and usage help

Read the README in any published language first, then search
existing issues. If the documentation does not answer your question, open a
bug report or feature request with the relevant version and invocation surface.

This repository does not currently operate a discussion forum or private
general-support channel. Maintainer availability is best-effort.

## Bugs and false positives

Use the issue forms:

- **Bug report** for crashes, incorrect behavior, or integration failures.
- **False positive / false negative** for a finding that should or should not
  have been reported.
- **Feature request** for a new user outcome or integration.

Use synthetic/minimal repositories where possible. Remove tokens, private URLs,
proprietary source, and personal data from reports and logs.

## Security and conduct

- Report vulnerabilities privately through [`SECURITY.md`](SECURITY.md).
- Report conduct incidents privately through [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md).
- Never publish live credentials, private repository content, or exploit
  details in an issue.
